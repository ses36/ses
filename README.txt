# শেষ নিউজ (Somoyer Alo)
এই প্রোজেক্টে একটি নিউজ ওয়েবসাইট তৈরি করা হয়েছে Google Sheets + Apps Script ব্যবহার করে।

## ফাইলসমূহ:
- index.html : হোমপেজ
- news.html : একক নিউজ ডিটেইলস পেজ
- assets/css/style.css : স্টাইলিং
- assets/js/script.js : Google Apps Script API থেকে নিউজ ফেচ করে
- assets/img/logo.svg : লোগো
- apps_script.gs : Google Apps Script ফাইল

## ডাটা সোর্স:
Google Form → Google Sheet → Apps Script → JSON → Website
